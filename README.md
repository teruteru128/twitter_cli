# cTwitter

## How to use

- `ctw <status>`
    - tweet &lt;status&gt;.
    - If no argument is specified, it reads standard input and tweets it.
- `ctw --register`
    - Register to ctw. 
- `ctw --login`
    - Login to ctw
- `ctw --logout`
    - Logout from ctw
- `ctw --init`
    - This may not be used.
- `ctw --tweet-length [status]`
    - Returns the length of the status.
    - Alias
        - `ctw -l [status]`
